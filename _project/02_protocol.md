RigoBlock is a complete execution stack: on-chain vault contracts, a safe off-chain engine that simulates every transaction before broadcast, and multiple interfaces for human and agent operators alike.

Vault operators delegate scoped permissions to AI agents, choose Manual or Delegated execution, and rely on NAV Shield to automatically block any trade that would reduce vault value beyond a configurable threshold. The execution engine operates outside the agent's control surface — it cannot be bypassed or disabled.

External agents connect through the X402 micropayment API, paying per call in USDC. No API keys, no subscriptions, no custody.

Deployed on Ethereum, Arbitrum, Base, Optimism, BSC, Polygon, and Unichain.
